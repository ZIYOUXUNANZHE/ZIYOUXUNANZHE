# 🐍 Python 在线学习计划

> 本计划适合零基础或有少量编程经验的学习者，目标是在 **6 个月**内系统掌握 Python，并能独立开发项目。

---

## 📅 学习总览

| 阶段 | 时间 | 内容 |
|------|------|------|
| 第一阶段 | 第 1–4 周 | Python 基础语法 |
| 第二阶段 | 第 5–8 周 | 面向对象与模块化编程 |
| 第三阶段 | 第 9–12 周 | 数据处理与文件操作 |
| 第四阶段 | 第 13–16 周 | Web 开发 / 数据分析方向 |
| 第五阶段 | 第 17–20 周 | 进阶主题与项目实战 |
| 第六阶段 | 第 21–24 周 | 综合项目与面试准备 |

---

## 🚀 第一阶段：Python 基础语法（第 1–4 周）

### 学习目标
- 搭建 Python 开发环境
- 掌握基本数据类型、运算符、控制流
- 熟悉函数与基本数据结构

### 每周安排

**第 1 周 – 环境与入门**
- [ ] 安装 Python 3.x 及 VS Code / PyCharm
- [ ] 变量、数据类型（int / float / str / bool）
- [ ] 输入输出（`input()` / `print()`）
- [ ] 算术、比较、逻辑运算符

**第 2 周 – 控制流**
- [ ] `if / elif / else` 条件语句
- [ ] `for` 循环与 `while` 循环
- [ ] `break` / `continue` / `pass`
- [ ] 列表（list）与元组（tuple）

**第 3 周 – 函数与字符串**
- [ ] 定义与调用函数
- [ ] 参数（位置参数、关键字参数、默认值）
- [ ] 返回值与作用域
- [ ] 字符串常用方法（`split` / `join` / `format` / f-string）

**第 4 周 – 字典与集合**
- [ ] 字典（dict）增删改查
- [ ] 集合（set）基本操作
- [ ] 列表推导式 / 字典推导式
- [ ] 综合练习：学生成绩管理小程序

### 推荐资源
- 🌐 [菜鸟教程 Python3](https://www.runoob.com/python3/python3-tutorial.html)
- 🌐 [Python 官方文档（中文）](https://docs.python.org/zh-cn/3/)
- 🎬 [B站：Python从入门到实践](https://search.bilibili.com/all?keyword=python%E5%85%A5%E9%97%A8)

---

## 🏗️ 第二阶段：面向对象与模块化编程（第 5–8 周）

### 学习目标
- 理解面向对象编程（OOP）思想
- 掌握类、继承与封装
- 学会使用标准库与第三方模块

**第 5 周 – 面向对象基础**
- [ ] 类（class）与对象
- [ ] `__init__` 构造方法
- [ ] 实例属性与类属性
- [ ] 实例方法、类方法、静态方法

**第 6 周 – 继承与多态**
- [ ] 单继承与多继承
- [ ] 方法重写（override）与 `super()`
- [ ] 魔术方法（`__str__` / `__repr__` / `__len__`）

**第 7 周 – 模块与包**
- [ ] `import` / `from … import`
- [ ] 标准库：`os` / `sys` / `math` / `random` / `datetime`
- [ ] 创建自己的模块与包
- [ ] `pip` 安装第三方库

**第 8 周 – 异常处理与调试**
- [ ] `try / except / else / finally`
- [ ] 自定义异常类
- [ ] 使用 `logging` 模块
- [ ] 调试技巧（断点 / `pdb`）

### 推荐资源
- 📖 《Python编程：从入门到实践》（第二版）第 9–11 章
- 🌐 [Real Python](https://realpython.com/)

---

## 📂 第三阶段：数据处理与文件操作（第 9–12 周）

### 学习目标
- 读写文件与处理 CSV / JSON
- 使用正则表达式
- 初步掌握 NumPy / Pandas

**第 9 周 – 文件 I/O**
- [ ] 读写文本文件（`open` / `with`）
- [ ] CSV 文件处理（`csv` 模块）
- [ ] JSON 序列化与反序列化

**第 10 周 – 正则表达式**
- [ ] `re` 模块基础（`match` / `search` / `findall`）
- [ ] 常用模式（字符类、量词、分组）
- [ ] 实战：从日志文件提取信息

**第 11 周 – NumPy 基础**
- [ ] 数组创建与索引
- [ ] 向量化运算与广播
- [ ] 常用数学函数

**第 12 周 – Pandas 基础**
- [ ] Series 与 DataFrame
- [ ] 数据读取（`read_csv` / `read_excel`）
- [ ] 数据清洗（缺失值 / 重复值）
- [ ] 分组聚合（`groupby`）

### 推荐资源
- 🌐 [Pandas 官方文档](https://pandas.pydata.org/docs/)
- 🎬 [NumPy & Pandas 视频教程（B站）](https://search.bilibili.com/all?keyword=pandas+python)

---

## 🌐 第四阶段：选择方向深入（第 13–16 周）

> 根据兴趣选择 **Web 开发** 或 **数据分析/可视化** 方向

### 方向 A：Web 开发（Flask / Django）

**第 13 周 – Flask 入门**
- [ ] 路由、视图函数、模板（Jinja2）
- [ ] GET / POST 请求处理

**第 14 周 – 数据库集成**
- [ ] SQLite + SQLAlchemy ORM
- [ ] 用户认证与 Session

**第 15–16 周 – 项目：个人博客系统**
- [ ] 文章的增删改查
- [ ] 用户注册 / 登录
- [ ] 部署到云服务器

### 方向 B：数据分析与可视化

**第 13 周 – Matplotlib / Seaborn**
- [ ] 折线图、柱状图、散点图、热力图
- [ ] 图表美化与导出

**第 14 周 – 数据分析实战**
- [ ] Kaggle 入门数据集（Titanic / Iris）
- [ ] 探索性数据分析（EDA）

**第 15–16 周 – 机器学习入门（Scikit-learn）**
- [ ] 线性回归 / 逻辑回归
- [ ] 决策树 / 随机森林
- [ ] 模型评估与交叉验证

### 推荐资源
- 🌐 [Flask 官方文档](https://flask.palletsprojects.com/zh_CN/)
- 🌐 [Kaggle Learn](https://www.kaggle.com/learn)
- 🌐 [Scikit-learn 官方文档](https://scikit-learn.org/)

## ⚡ 第五阶段：进阶主题（第 17–20 周）

**第 17 周 – 迭代器与生成器**
- [ ] `iter()` / `next()`
- [ ] `yield` 关键字与生成器函数
- [ ] `itertools` 模块

**第 18 周 – 装饰器与高阶函数**
- [ ] 闭包与作用域
- [ ] 装饰器原理与应用
- [ ] `functools`：`lru_cache` / `partial`

**第 19 周 – 并发编程**
- [ ] 多线程（`threading`）
- [ ] 多进程（`multiprocessing`）
- [ ] 异步编程（`asyncio` / `await`）

**第 20 周 – 测试与代码质量**
- [ ] 单元测试（`unittest` / `pytest`）
- [ ] 代码风格（PEP 8 / `flake8` / `black`）
- [ ] 类型注解（`typing`）

## ✅ 每日学习建议

- ⏰ **每天至少 1 小时**的学习时间
- 💻 **边学边练**：看完每个知识点立即写代码验证
- 📝 **做笔记**：用 Markdown 记录重要概念和代码片段
- 🔄 **复盘**：每周末回顾本周所学，巩固薄弱环节
- 🤝 **交流**：加入 Python 学习群，积极提问和分享
